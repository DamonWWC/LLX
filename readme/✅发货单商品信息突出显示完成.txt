✅ 发货单商品信息突出显示完成
================================

功能概述
--------
对发货单中的商品信息进行了视觉优化，使用显著的颜色和字体大小
突出显示商品名称和数量，提升发货单的可读性和实用性。

主要改进
--------

### 1. 商品信息一行显示
- **字体大小**：20px 加粗字体
- **颜色**：使用 **#1890ff**（蓝色）突出显示
- **格式**：1.稻花香（10斤/箱）数量：2箱
- **布局**：所有信息在一行显示，简洁明了

### 2. 信息整合优化
- **包含内容**：序号、商品名、规格、数量
- **去除冗余**：不再单独显示规格信息
- **紧凑布局**：减少行数，提升可读性

### 3. 总计信息优化
- **商品总数**：
  - 字体：22px 加粗
  - 颜色：#ff6034（橙色）
  - 突出显示总数量

- **商品总重**：
  - 字体：22px 加粗
  - 颜色：#52c41a（绿色）
  - 突出显示总重量

颜色方案
--------
- **蓝色 (#1890ff)**：商品信息（一行显示）
- **橙色 (#ff6034)**：商品总数
- **绿色 (#52c41a)**：商品总重
- **黑色 (#333333)**：标题信息

布局优化
--------
1. **商品信息一行显示**：
   - 格式：1.稻花香（10斤/箱）数量：2箱
   - 字体：20px 加粗蓝色
   - 包含：序号、商品名、规格、数量

2. **间距调整**：
   - 商品之间间距：15px
   - 行高：25px
   - 简洁紧凑的布局

3. **信息整合**：
   - 所有商品信息在一行显示
   - 去除冗余的规格单独显示
   - 保持重要信息的突出显示

视觉效果
--------
✓ 商品信息一行显示，简洁明了
✓ 蓝色大字体突出显示商品信息
✓ 数量信息清晰可见
✓ 布局紧凑，节省空间
✓ 整体布局美观，专业感强

使用场景

--------
- 使用 Canvas 2D API 绘制
- 动态计算字体大小和颜色
- 支持多商品订单显示
- 自动换行处理长商品名称
