### 1. 电商标签换行格式 ✨

```

### 2. 遮蔽手机号智能识别 🔐

✅ 支持格式：
```

所在地区: 湖北省荆州市松滋市新江口街道

✅ 特点：
• 自动检测遮蔽号码（包含星号 ****）

• 其他信息正常识别填充
• 只需手动补全手机号（节省80%时间）
