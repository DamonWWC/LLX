=====================================
         完整购物流程演示
=====================================

第1步：浏览商品
  → 打开小程序，自动进入"商品"页面
  → 看到默认的大米商品列表
  → 每个商品显示图片、名称、价格、运费

第2步：添加自定义商品（可选）
  → 点击右上角"+"按钮
  → 填写商品信息（名称、价格、运费）
  → 点击"上传图片"选择相册或拍照
  → 确认添加，新商品出现在列表中

第3步：选购商品
  → 点击商品卡片的"+"按钮增加数量
  → 右下角购物车图标显示商品总数
  → 可以点击"-"按钮减少数量

第4步：查看购物车
  → 底部显示"去结算"按钮
  → 购物车右上角数字显示商品数量

第5步：结算
  → 点击"去结算"按钮
  → 弹出结算详情面板
  → 显示：商品明细、价格汇总

第6步：选择收货地址
  → 点击"选择收货地址"
  → 跳转到地址管理页面
  → 选择现有地址 或 添加新地址

第7步：添加地址（首次使用）
  → 点击"添加新地址"
  → 方式1：手动填写表单
  → 方式2：点击"智能粘贴地址"自动识别
  → 保存地址

第8步：确认订单
  → 返回结算页面
  → 显示选中的收货地址
  → 确认商品明细和总价
  → 点击"确认下单"

第9步：下单成功
  → 显示"下单成功"提示
  → 购物车自动清空
  → 提示"可在我的页面查看订单"

第10步：查看订单
  → 切换到"我的"Tab
  → 看到刚才的订单
  → 点击"查看详情"查看完整信息

第11步：管理订单和地址
  → 在订单列表中删除订单
  → 或点击"清空订单"清空所有
  → 点击"收货地址"管理地址

=====================================
           核心技术要点
=====================================

1. TabBar 配置
   - app.json 中配置 tabBar
   - 两个页面：index（商品）、my（我的）
   - 使用 wx.switchTab 切换

2. 数据存储
   - riceProducts - 商品数据
   - orderList - 订单列表
   - shippingAddresses - 地址列表
   - 全部使用 wx.setStorageSync 本地存储

3. 订单生成
   - 时间戳作为订单ID
   - 订单号：ORD + 时间戳
   - 包含商品、地址、金额、时间

4. 页面导航
   - TabBar页面：wx.switchTab
   - 普通页面：wx.navigateTo
   - 地址选择：带参数传递

5. 数据同步
   - 订单页面 onShow 时刷新数据
   - 本地存储自动保存
   - 实时更新显示
