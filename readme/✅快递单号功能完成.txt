   - 已发货且有快递单号的订单会显示快递信息

   - 显示发货时间

   - 在"更多"菜单中添加快递单号编辑选项

   - 支持清除快递单号

4. ✅ 数据结构支持
   - 订单对象新增 trackingNumber 字段存储快递单号
   - 订单对象新增 shippingTime 字段存储发货时间
   - 数据持久化到本地存储

技术实现细节：

📱 用户交互流程：
1. 用户点击"确认发货" → 弹出输入框 → 可选择输入快递单号 → 确认发货
2. 已发货订单在列表显示快递信息（如果有）
3. 点击"更多" → "修改快递单号"/"添加快递单号" → 编辑快递单号

- 快递信息区域：浅灰色背景，圆角设计

- 发货时间：灰色小字显示
- 编辑按钮：蓝色主题，区分删除操作

💾 数据管理：
- 使用 wx.setStorageSync 持久化数据
- 实时更新页面显示
