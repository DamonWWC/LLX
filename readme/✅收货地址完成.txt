  🎯 使用流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【添加地址】
  方式一：手动输入
    1. 点击"添加新地址"
    2. 填写收件人、电话等信息
    3. 点击"保存"

  方式二：智能识别
    1. 点击"智能识别粘贴"
    2. 粘贴完整地址信息
    3. 点击"智能识别"
    4. 检查并保存

【管理地址】
    ✓ 编辑：修改地址信息
    ✓ 删除：移除不需要的地址
    ✓ 设为默认：设置常用地址

【结算下单】
    1. 选择商品，加入购物车
    2. 点击"立即结算"
    3. 查看结算详情
    4. 点击"选择收货地址"
    5. 选择或添加地址
    6. 确认订单信息
    7. 点击"确认下单"
    8. 完成订单

  • 复杂地址建议手动输入

默认地址：
  • 首个地址自动设为默认
  • 只能有一个默认地址
  • 设置新默认会取消旧默认

数据安全：
  • 所有数据保存在本地
  • 卸载小程序会丢失
  • 建议定期截图备份
