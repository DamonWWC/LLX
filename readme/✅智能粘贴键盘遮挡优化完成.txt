✅ 智能粘贴键盘遮挡优化完成

问题描述：
- 在添加收货地址时，点击智能粘贴按钮
- 当手机键盘弹起时，会遮挡粘贴地址信息框
- 影响用户输入和查看粘贴内容

解决方案：

1. 📱 键盘高度监听
   - 使用 wx.onKeyboardHeightChange API 监听键盘高度变化
   - 实时获取键盘高度并更新页面数据

2. 📱 动态调整弹窗位置
   - 智能粘贴弹窗根据键盘高度动态调整 bottom 位置
   - 键盘弹起时：bottom = keyboardHeight
   - 键盘收起时：bottom = 50px

3. 📱 弹窗布局优化
   - 设置弹窗最大高度为 80vh，避免超出屏幕
   - 内容区域支持滚动，确保长文本可查看
   - 输入框设置最大高度限制，防止过度拉伸

4. 📱 用户体验改进
   - 添加过渡动画，位置变化更平滑
   - 输入框聚焦时高亮显示
   - 保持原有的智能识别功能不变

技术实现：

JavaScript 部分：
- 添加 keyboardHeight 数据字段
- 在 onLoad 中设置键盘监听
- 在 onUnload 中清理监听器

WXML 部分：
- 弹窗添加动态 style 属性
- 根据键盘高度调整 bottom 位置

WXSS 部分：
- 弹窗添加 transition 过渡效果
- 内容区域支持滚动
- 输入框添加焦点样式

效果展示：
✓ 键盘弹起时，粘贴弹窗自动上移到键盘上方
✓ 键盘收起时，粘贴弹窗回到底部位置
✓ 输入框始终可见，不被键盘遮挡
✓ 长文本内容可以滚动查看
✓ 操作流畅，用户体验良好

兼容性：
- 支持所有微信小程序版本
- 自动适配不同屏幕尺寸
- 响应式布局，适配各种设备
