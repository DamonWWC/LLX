✅ 直辖市地址识别优化完成

问题描述：
- 对于直辖市地址，省份一栏应该去掉"市"字
- 如"北京市"，省份填写"北京"，城市填写"北京市"
- 提高地址识别的准确性和规范性

解决方案：

1. 📱 直辖市特殊处理逻辑
   - 识别直辖市：北京、天津、上海、重庆
   - 省份字段：去掉"市"字（如：北京、上海、天津、重庆）
   - 城市字段：保留完整名称（如：北京市、上海市、天津市、重庆市）

2. 📱 技术实现
   - 修改了 `extractProvinceCityDistrictEnhanced` 函数
   - 在省份匹配规则中添加 `type: 'municipality'` 标识
   - 对直辖市进行特殊处理，确保省份和城市字段正确分离

3. 📱 处理逻辑
   ```javascript
   // 直辖市：省份去掉"市"字，城市设为完整名称
   if (rule.type === 'municipality') {
     province = provinceName;  // 去掉"市"字
   }
   
   // 直辖市特殊处理：确保城市名称正确
   if (province && MUNICIPALITIES.includes(province)) {
     if (!city) {
       city = province + '市';  // 城市设为完整名称
     }
   }
   ```

4. 📱 测试用例
   - 北京市朝阳区xxx街道 → 省份：北京，城市：北京市
   - 上海市浦东新区xxx路 → 省份：上海，城市：上海市
   - 天津市和平区xxx街 → 省份：天津，城市：天津市
   - 重庆市渝中区xxx道 → 省份：重庆，城市：重庆市

效果展示：
✓ 直辖市省份字段规范：不包含"市"字
✓ 直辖市城市字段完整：包含"市"字
✓ 非直辖市地址不受影响：保持原有逻辑
✓ 地址识别准确性提升：符合标准格式

用户体验：

- 避免省份和城市字段重复
- 符合中国行政区划标准
- 提升地址识别的专业性
