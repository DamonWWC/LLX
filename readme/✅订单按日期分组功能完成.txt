订单按日期分组功能实现完成

📅 功能概述
- 订单管理页面现在按照订单创建日期进行分组显示
- 每个日期组可以独立折叠和展开
- 支持智能日期显示（今天、昨天、具体月日）
- 保持原有的搜索、过滤、多选等功能

🔧 技术实现

1. 数据结构优化
   - 新增 groupedOrderList 数组存储分组后的订单
   - 新增 collapsedDates 对象管理折叠状态
   - 保持原有 filteredOrderList 用于搜索过滤

2. 日期分组逻辑
   - groupOrdersByDate() 函数实现订单按日期分组
   - 支持多种时间格式解析（空格分隔、ISO格式等）
   - 按日期倒序排列，最新日期在前

3. 智能日期显示
   - formatDisplayDate() 函数格式化日期显示
   - 今天显示为"今天"
   - 昨天显示为"昨天"  
   - 其他日期显示为"X月X日"
   - 无法解析的日期显示为"未知日期"

4. 折叠交互功能
   - toggleDateGroup() 函数处理折叠/展开
   - 点击日期头部可切换折叠状态
   - 折叠状态持久化保存
   - 平滑的动画过渡效果

5. 界面优化
   - 日期分组头部采用渐变背景
   - 显示日期和订单数量
   - 折叠图标带动画效果
   - 订单项在分组内样式调整
