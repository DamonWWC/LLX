问题原因
--------
之前使用了外部图片服务 via.placeholder.com，
在某些网络环境下（如中国大陆）无法访问，
导致错误：ERR_NAME_NOT_RESOLVED

解决方案
--------
✅ 已将所有外部图片URL替换为本地SVG图片
✅ 使用 data URI 格式，无需网络请求
✅ 完全离线可用

修改内容
--------
1. 默认商品图片（3个）
   - 东北大米：绿色背景 + 🌾 emoji
   - 泰国香米：橙色背景 + 🌾 emoji
   - 五常稻花香：粉色背景 + 🌾 emoji

2. 添加商品时的默认图片
   - 灰色背景 + 🌾 emoji

3. 默认图片选择器（5个主题）
   - 绿色主题
   - 橙色主题
   - 粉色主题
   - 蓝色主题
   - 灰色通用

图片技术说明
-----------
使用 SVG data URI 格式：
- 格式：data:image/svg+xml,%3Csvg...%3E
- 优点：无需网络，完全本地化
- 内容：纯色背景 + emoji图标
- 大小：约300×300像素

使用方法
--------
1. 编译运行小程序
2. 图片将自动显示（无需网络）
3. 用户可以点击"上传图片"替换为真实图片
4. 或点击"使用默认图片"选择其他主题

清除旧数据（如果需要）
