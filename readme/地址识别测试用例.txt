地址识别测试用例 - 直辖市特殊处理

测试场景：
对于直辖市地址，省份一栏去掉"市"字，城市填写完整名称

测试用例：

1. 北京市朝阳区xxx街道xxx号
   期望结果：
   - 省份：北京
   - 城市：北京市
   - 区县：朝阳区

2. 张三 13800138000 北京市海淀区中关村大街1号
   期望结果：
   - 姓名：张三
   - 手机号：13800138000
   - 省份：北京
   - 城市：北京市
   - 区县：海淀区
   - 详细地址：中关村大街1号

3. 上海市浦东新区陆家嘴环路1000号
   期望结果：
   - 省份：上海
   - 城市：上海市
   - 区县：浦东新区
   - 详细地址：陆家嘴环路1000号

4. 天津市和平区南京路123号
   期望结果：
   - 省份：天津
   - 城市：天津市
   - 区县：和平区
   - 详细地址：南京路123号

5. 重庆市渝中区解放碑步行街456号
   期望结果：
   - 省份：重庆
   - 城市：重庆市
   - 区县：渝中区
   - 详细地址：解放碑步行街456号

6. 李四 13900139000 北京市西城区西单北大街789号
   期望结果：
   - 姓名：李四
   - 手机号：13900139000
   - 省份：北京
   - 城市：北京市
   - 区县：西城区
   - 详细地址：西单北大街789号

技术实现：
- 修改了 extractProvinceCityDistrictEnhanced 函数
- 对直辖市进行特殊处理逻辑
- 省份匹配时去掉"市"字
- 城市自动设置为完整名称（如"北京市"）

验证方法：
1. 在地址管理页面点击"智能识别粘贴"
2. 输入上述测试用例中的地址信息
3. 点击"智能识别"按钮
4. 检查解析结果是否符合期望

注意事项：
- 直辖市包括：北京、天津、上海、重庆
- 省份字段不包含"市"字
- 城市字段包含完整的"市"字
- 其他省份和城市不受影响
