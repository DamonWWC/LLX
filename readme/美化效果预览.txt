🎬 动画效果一览

[商品卡片]
  点击时 → 向上浮起 4rpx + 阴影加深
  
[加减按钮]
  点击时 → 缩小到 0.9 倍
  
[删除按钮]
  点击时 → 旋转 90° + 变红色
  
[结算面板]
  弹出时 → 弹簧式滑入（有回弹感）
  
[关闭按钮]
  点击时 → 旋转 90° 消失
