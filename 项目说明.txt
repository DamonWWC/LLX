完整购物流程
-----------
1. 【商品页】选择商品 → 调整数量 → 查看购物车
2. 【结算】点击去结算 → 查看明细 → 选择地址
3. 【地址】选择现有地址 / 添加新地址 / 智能粘贴
4. 【下单】确认信息 → 确认下单 → 订单生成
5. 【我的】切换到我的页面 → 查看订单列表
6. 【管理】查看订单详情 / 删除订单 / 管理地址

数据存储
--------
所有数据使用本地存储（wx.setStorageSync / wx.getStorageSync）：

1. riceProducts - 商品数据
   - 包含商品名称、价格、运费、图片、数量

2. orderList - 订单列表
   - 订单号、状态、商品明细
   - 收货地址、金额、创建时间

3. shippingAddresses - 收货地址
   - 姓名、电话、省市区、详细地址
   - 是否默认地址

技术实现
--------
- 微信小程序原生框架
- TabBar 底部导航
- 购物车数据管理
- Canvas 2D API 生成图片
- 本地存储 (wx.setStorageSync, wx.getStorageSync)
- 图片上传 (wx.chooseImage)
- 智能地址解析 (正则表达式)
- 页面导航 (wx.navigateTo, wx.switchTab)
- CSS3 动画和交互反馈

TabBar 图标说明
--------------
✅ 当前使用纯文字导航模式
- 底部显示"商品"和"我的"文字
- 未选中：灰色 (#999999)
- 选中：橙红色 (#ff6034)
- 无需图标文件，立即可用

如需添加图标：
📖 详细教程：查看《如何添加TabBar图标.md》
1. 从图标网站下载（iconfont.cn、iconpark.bytedance.com）
2. 尺寸：81px × 81px
3. 格式：PNG，透明背景
4. 颜色：未选中-灰色(#999)，选中-红色(#ff6034)
5. 放到 images/ 目录
6. 修改 app.json 添加图标路径
7. 重新编译
